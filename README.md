## Develop

### Git branch

開発・メンテナンスを容易にするためGitHub Flowを採用しています。  
ブランチ名は``feature-``から始まる名前で``main``ブランチから切ってください。

### Git commit

#### Template

```
<type>: <subject>
```

#### Type

- **feat**: 新機能
- **change**: 修正・削除
- **fix**: バグフィックス
- **docs**: ドキュメントに関する変更
- **style**: フォーマット等の変更
- **refactor**: リファクタに関する変更
- **debug**: デバック用のコード
